# Lista PHP 01

Repositório com atividades de PHP puro aplicados com o prof Icaro Botelho.

## Execução dos códigos

Abra os arquivos em um servidor local com PHP, como o XAMPP, e acesse os exercícios pelo navegador.

## Exercícios

- `ex01.php` - Calculadora de engenharia
- `ex02.php` - Espelho mágico
- `ex03.php` - Cadastro seguro
- `ex04.php` - Gerador de senha
- `ex05.php` - Estatísticas de texto
- `ex06.php` - Conversor de temperatura
- `ex07.php` - sistema de desconto
- `ex07.php` - Sistema de Descontos
- `ex08.php` - Organizador de Lista
- `ex09.php` - Verificador Matemático
- `ex10.php` - Sistema de Notas
- `ex11.php` - Formatador de Relatórios
- `ex12.php` - Catálogo de Produtos
- `ex13.php` - Criptografia Simples
- `ex14.php` - Estatísticas Numéricas
- `ex15/` - Biblioteca de Funções

## Exercício 15

O exercício 15 está separado em 2 arquivos:

- `ex15/funcoes.php` com as funções reutilizáveis
- `ex15/index.php` com a demonstração de uso

Para testar, basta abrir o `index.php` do exercício no navegador.
